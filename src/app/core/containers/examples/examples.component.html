Dialog

We use the Angular CDK Dialog in combination with our own Dialog Component which handles the html & styling of the dialog itself.
You can open a Dialog from a component or a template. In this example page we use a template.
For more information on the Dialog Options go to the Docs: https://material.angular.io/cdk/dialog/api

The Dialog component has a couple of options:

Title
Title of the dialog (required)
Size
Normal (default), small or large
Content (ng-content)
The main content of the dialog
Footer content - Left (footer-left)
Buttons/loader/etc
Footer content - Right (footer-right)
